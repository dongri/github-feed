# Github Feed
Chrome extension to display old github feeds

# How to use
1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable `Developer mode`
4. Click `Load unpacked`
5. Select the folder of this repo
6. Go to `https://github.com`
